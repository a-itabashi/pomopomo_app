# 卒業課題

# アプリ名：pomopomo(ポモポモ)

## 概要
ポモドーロ・テクニック(25分作業・５分休憩のサイクル)を活用した作業集中アプリ。

## コンセプト
作業中に音楽を聞くことも可能。掲示板機能もあり。  
PCで作業をする全ての人をターゲットとしている。

## バージョン
- Ruby 2.5.5
- Rails 5.2.3

## 機能一覧
- ログイン機能/ユーザー登録機能
- マイページ
- タイマー機能
- 音楽選択/再生機能
- 掲示板機能
- 管理者権限
- 辞書機能

### ※優先度が低い機能一覧
- 学習管理機能(カレンダー機能)
- 聞いた音楽を一覧で表示させる機能(検索機能も)
- Twitterのツイートボタン機能
